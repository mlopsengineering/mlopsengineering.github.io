About

The goal of the project is to provide a way to keep track of the latest advancements in artificial intelligence for machine learning practitioneers in a weekly, digestable email mailing list.

We built an API to monitor 1000s of data sources including blogs, open access journals and articles while filtering out low quality feeds. Our project runs in a cloud VM as a Jupyter notebook and we are continuously iterating to improve the newsletter.

Our project is unique because we also update an index called the "singularity clock", a infographic that is part of each issue of our newsletter and measures the estimated time to singularity.

This metric is reduced each time a major advance happens such as the discovery of a new model architecture or statistical methodology that moves the field forward.
